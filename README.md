### require: python3

# csv2xlsx

#### Convert csv file to xlsx file using python

<br />

## How to Run
```shell
python3 csv2xlsx.py
```
(if pip install error, run below command)
```shell
sudo python3 csv2xlsx.py
```

<br />

2. input csv file name(don't forget adding .csv)
